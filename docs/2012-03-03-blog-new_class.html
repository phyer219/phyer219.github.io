分班

2012-03-03

-blog

好久没回家了。一个月吧，自打开学。

喜欢妹妹，越来越讨人喜欢。

对于我那个同桌，没得说。

这学期，分班，也就和一些人少有联系。准确地说，21班没有几个人能说说话，蔡和李还是很好的，教我弹吉它。是好朋友。分班以后，我便归于平静，少言。 我的生活，就是学习，看书，口琴，吉它，去图书馆。其实这样也是不错的，感觉是充实的。 我已经习惯一个人在大街上走。周末，走去图书馆，然后翻一一些书籍，一站便是四五个小时，回宿舍的床上一躺，也是十分惬意！我希望，这种平静而又充实的生活，会给我一个好的结果，至少 是实现我的梦想。 以前也会在宿舍听音乐的，不过耳机坏了，也就少听，偶尔听下广播。新分了宿舍，虽然大家都很HUANG，让我长了不少见识，大家去也是相处得很好，还算不错嘛！总之，这学期是前所未有的一种体验。
